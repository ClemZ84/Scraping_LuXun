# Scraping-Lu-Xun
一个自用的小小爬虫脚本，能够爬取北京鲁迅博物馆中的鲁迅书信和日记

## 功能
1. 爬取北京鲁迅博物馆数据库中的鲁迅书信和日记：http://www.luxunmuseum.com.cn/cx/works.php  
2. 将这些书信和日记输出为.txt文件的纯文本格式  
3. 标题格式分别为：  
   书信：[年.月.日_通信人] e.g. 1904.10.08_蒋抑卮  
   日记：[年.月] e.g. 1912.11

## 说明
由于个人暂时用不上，所以没做爬取“著作”和“专题”的脚本，希望有大佬可以顺手做一个完整版  
使用前需要填入自己浏览器的user-agent和输出路径
